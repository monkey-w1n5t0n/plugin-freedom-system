## Context to Preserve

**Key Decisions:**
- Plugin type: Audio Effect (Tape Saturator)
- Core concept: Warm tape saturation with controllable degradation (wow/flutter/dropout/noise)
- AGE parameter must remain musical even at maximum—never "stupid" sounding
- Visual aesthetic: Premium vintage tape machine, skeuomorphic brass controls, earth tones
- 3-parameter design: DRIVE (saturation), AGE (degradation), MIX (dry/wet)

**Design Details (v2):**
- Window: 500x450px (vertically centered content block)
- VU meter: Classic analog scale (-20 to +3 dB), bronze frame, glass effect, animated needle
- Title: "TAPE AGE" with 0.35em letter spacing, embossed text shadow
- Knobs: 90px diameter brass with knurled edges (48 ridges), brushed metal center, rotating indicator line
- Labels: Directly below knobs (DRIVE, AGE, MIX), no percentage displays
- Colors: Creamy beige background (#ece6d9), richer brass (#b8894a), deep brown accents
- Textures: Fabric weave overlay, subtle vignette, premium shadows throughout

**Technical Notes:**
- WebView-based UI (JUCE WebBrowserComponent)
- JavaScript ↔ C++ communication via `window.juce.postMessage()`
- Parameter bindings handle automation sync
- VU meter updates at 60 Hz via timer callback
- All parameters normalized 0.0-1.0 (JUCE convention)
